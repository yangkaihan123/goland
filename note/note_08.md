**常量**
相对于变量，常量是恒定不变的值，多用于定义程序运行期间不会改变的那些值  
常量的声明和变量的声明非常类似，var换成了const, 常量在定义的时候必须赋值  
```go
const pi = 3.1415
const e = 2.7182
```   
声明了pi和e 这两个常量后，在整个程序的运行期间，它们的值都不能再发送变化  
多常量一起声明：  
```go
const (
	pi = 3.1415
	e = 2.7182
)
```   
const 同时声明多个常量时，如果省略了值则表示和上面一行值相同：  
```go
const (
    n1 = 100
	n2
	n3
)
```   
n1,n2,n3 值都是100
